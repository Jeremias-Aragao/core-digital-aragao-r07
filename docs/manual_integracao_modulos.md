# Manual de integração de módulos R07

## Checklist
1. Criar pasta do módulo em `app/modules/nome/`
2. Criar `routes.py`
3. Criar template em `app/templates/modules/nome/`
4. Criar static isolado em `app/static/modules/nome/`
5. Registrar blueprint em `app/__init__.py`
6. Cadastrar módulo na tabela `modules`
7. Definir cores próprias se quiser identidade visual específica
