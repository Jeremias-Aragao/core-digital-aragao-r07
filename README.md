# Core Digital Aragão R07

R07 evolui a arquitetura para suportar módulos internos com identidade visual própria, mantendo o Core como hub central.

## Entregas
- Core público premium
- admin separado
- área do cliente
- módulo financeiro interno
- módulo portfólio interno
- tema por módulo (cores próprias)
- gerenciador de módulos com cores e modo do módulo
